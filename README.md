## How to use

### 1. Clone the repository
```bash
conda env create -n whoisplaying -f environment.yml
conda activate whoisplaying
```

### 2. Run the script
```bash
python main.py
```
Optional arguments:
- `--teams` or `-t`: to add any other teams
- `--urls` or `-d`: to add the accompanying urls
